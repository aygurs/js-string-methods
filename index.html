JavaScript String Methods

variable.charAt()

This method will return the character at the index number put between the brackets.

The syntax for this method looks like this: word.charAt(indexNumber).

Remember - computers count from 0! So if the index number is 2, it will count: 0, 1, 2 (Meaning you will get the 3rd character).

If charAt() cannot perform its function, it will return an empty string. Try to find what causes an error!
